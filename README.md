# IBD Microbiome Statistical Modeling

This repository contains reproducible R workflows for statistical modeling of public gut microbiome data in the context of inflammatory bowel disease (IBD).

The project is currently under development.

## Planned analyses

- Data loading from public microbiome resources
- Exploratory data analysis
- Logistic regression for IBD status prediction
- Feature selection and model interpretation
- ROC/AUC-based performance evaluation
- Critical discussion of limitations such as compositionality, batch effects, and cohort structure

## Purpose

The goal of this repository is educational and methodological: to demonstrate reproducible statistical modeling workflows for microbiome data using R.
